How to Use Painter Pro

🎨 Getting Started

1. Load a 3D model using the “Load 3D Model” button

2. Click on any part of the model to select it

3. Choose a color or texture to apply

🖱️ Camera Controls

Rotate: Right-click + Drag

Pan: Middle-click + Drag or Shift + Right-click + Drag

Zoom: Mouse Scroll Wheel

🎯 Selection Mode

Click an object to enter selection mode and apply materials.

Press Esc to deselect and return to camera movement.

📱 Touch Controls (Mobile/Tablet)

Rotate: One finger drag

Pan: Two finger drag

Zoom: Pinch gesture

📸 Download 4K

Click the camera button in the header to download a high-resolution PNG of the current view.

💡 Tips

• Use the search box to quickly find colors or textures

• Material controls include Tiling, Roughness, and Metallic

• Toggle between light and dark themes for comfort

• Reset materials anytime with the Reset button
